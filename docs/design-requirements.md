## 1. プロジェクト概要

### Nuform とは？

**ニューモフィズム2.0** デザインに基づいたモダンなReact UIコンポーネントライブラリ。

従来のニューモフィズムの問題点（低コントラスト、アクセシビリティの課題）を解決し、実用的で美しいデザインを実現。

### ターゲットユーザー

- React/Next.js開発者
- モダンでおしゃれなUIを求める人
- アクセシビリティを重視する開発者

### 競合・参考ライブラリ

- shadcn/ui
- Radix UI
- Chakra UI
- MUI

## 2. ニューモフィズム2.0 デザインの特徴

### 基本コンセプト

ニューモフィズム（Neumorphism）は、要素が背景から「浮き出ている」または「凹んでいる」ように見せるデザインスタイル。

**2.0の改善点:**
- 高コントラストでテキストが読みやすい
- アクセシビリティ基準（WCAG）を満たす
- インタラクションがわかりやすい

### シャドウシステム

```
浮き出し（Elevation）:
  - 明るいシャドウ（左上）+ 暗いシャドウ（右下）
  - ホバーでより浮き上がる
  - クリックで凹む（押し込まれる感覚）

凹み（Inset）:
  - Input、選択されたタブなどに使用
  - 「押されている」状態を表現
```

### インタラクション

| 状態 | 効果 |
|-----|------|
| デフォルト | 軽く浮き出し |
| ホバー | より浮き上がる + わずかに上移動 |
| アクティブ/クリック | 凹む（押し込まれる） |
| フォーカス | フォーカスリング表示 |
| 無効 | 低opacity、シャドウなし |

## 3. カラーパレット

### ライトモード

```css
/* 背景 */
--nuform-bg: #e0e5ec;          /* メイン背景（ソフトグレー） */

/* テキスト */
--nuform-text: #2d3748;         /* メインテキスト（ダークグレー） */
--nuform-text-muted: #718096;   /* サブテキスト */

/* アクセント */
--nuform-primary: #6366f1;      /* プライマリ（インディゴ） */
--nuform-primary-hover: #4f46e5;
--nuform-success: #22c55e;      /* 成功（グリーン） */
--nuform-warning: #f59e0b;      /* 警告（オレンジ） */
--nuform-error: #ef4444;        /* エラー（レッド） */
--nuform-info: #3b82f6;         /* 情報（ブルー） */

/* シャドウ */
--nuform-shadow-light: rgba(255, 255, 255, 0.8);  /* 明るいシャドウ */
--nuform-shadow-dark: rgba(163, 177, 198, 0.6);   /* 暗いシャドウ */
```

### ダークモード

```css
/* 背景 */
--nuform-bg-dark: #1a1a2e;

/* テキスト */
--nuform-text-dark: #e2e8f0;

/* シャドウ */
--nuform-shadow-light-dark: rgba(45, 45, 70, 0.3);
--nuform-shadow-dark-dark: rgba(10, 10, 20, 0.5);
```

## 4. 既存コンポーネント一覧

### Button
- バリアント: `default`, `primary`, `ghost`
- サイズ: `sm`, `md`, `lg`
- 機能: ローディング状態、アイコン対応、無効状態

### Card
- バリアント: `elevated`（浮き出し）, `flat`（フラット）, `inset`（凹み）
- パディング: `none`, `sm`, `md`, `lg`

### Input
- 凹んだデザインの入力フィールド
- ラベル、エラーメッセージ対応
- フォーカス時のスタイル

### Avatar
- サイズ: `sm`, `md`, `lg`, `xl`
- 画像またはイニシャル表示

### Badge
- バリアント: `default`, `success`, `warning`, `error`, `info`
- サイズ: `sm`, `md`, `lg`

### Divider
- 方向: `horizontal`, `vertical`
