# インストール

Neumo UIをプロジェクトに追加する方法を説明します。

## パッケージのインストール

お好みのパッケージマネージャーでインストールしてください。

```bash tab="npm"
npm install neumo-ui
```

```bash tab="pnpm"
pnpm add neumo-ui
```

```bash tab="yarn"
yarn add neumo-ui
```

## スタイルのインポート

Neumo UIのデザイントークン（CSS変数）を使用するため、スタイルをインポートしてください。

```tsx
// アプリケーションのエントリーポイント（例: App.tsx, main.tsx）
import 'neumo-ui/styles.css';
```

## 使用例

```tsx
import { Button, Card, Input, Badge, Avatar, Divider } from 'neumo-ui';
import 'neumo-ui/styles.css';

function App() {
  return (
    <div style={{ background: 'var(--neumo-bg)', minHeight: '100vh', padding: '2rem' }}>
      <Card>
        <div style={{ display: 'flex', alignItems: 'center', gap: '1rem', marginBottom: '1rem' }}>
          <Avatar alt="User" size="lg" />
          <div>
            <h2>Welcome back!</h2>
            <Badge variant="success">Active</Badge>
          </div>
        </div>
        
        <Divider />
        
        <Input label="Email" type="email" placeholder="Enter your email" />
        
        <div style={{ marginTop: '1rem', display: 'flex', gap: '0.5rem' }}>
          <Button variant="primary">Submit</Button>
          <Button variant="ghost">Cancel</Button>
        </div>
      </Card>
    </div>
  );
}
```

## 背景色の設定

ニューモフィズムデザインは背景色が重要です。`--neumo-bg` 変数を使用して背景を設定してください。

```css
body {
  background: var(--neumo-bg);
}
```

## TypeScript

Neumo UIはTypeScriptで書かれており、型定義が含まれています。追加の型パッケージは不要です。

```tsx
import type { ButtonProps, CardProps, InputProps } from 'neumo-ui';
```

## 要件

- React 18以上
- React DOM 18以上
