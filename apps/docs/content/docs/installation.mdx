# インストール

Nuformをプロジェクトに追加する方法を説明します。

<Callout type="info" title="動作環境">
- **React**: 18.0以上（React 19推奨）
- **TypeScript**: 5.0以上（推奨）
- **Node.js**: 18.0以上
</Callout>

## セットアップ手順

<Steps>
  <Step>
    ### パッケージをインストール

    お好みのパッケージマネージャーでインストールしてください。

    <Tabs items={['npm', 'pnpm', 'yarn', 'bun']} persist groupId="pm">
      <Tab value="npm">
        ```bash
        npm install @anju/nuform
        ```
      </Tab>
      <Tab value="pnpm">
        ```bash
        pnpm add @anju/nuform
        ```
      </Tab>
      <Tab value="yarn">
        ```bash
        yarn add @anju/nuform
        ```
      </Tab>
      <Tab value="bun">
        ```bash
        bun add @anju/nuform
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### スタイルをインポート

    Nuformのデザイントークン（CSS変数）を使用するため、アプリケーションのエントリーポイントでスタイルをインポートしてください。

    <Tabs items={['Next.js (App Router)', 'Next.js (Pages Router)', 'Vite / CRA']} persist groupId="framework">
      <Tab value="Next.js (App Router)">
        ```tsx title="app/layout.tsx"
        import '@anju/nuform/styles.css';

        export default function RootLayout({ children }) {
          return (
            <html lang="ja">
              <body>{children}</body>
            </html>
          );
        }
        ```
      </Tab>
      <Tab value="Next.js (Pages Router)">
        ```tsx title="pages/_app.tsx"
        import '@anju/nuform/styles.css';
        import type { AppProps } from 'next/app';

        export default function App({ Component, pageProps }: AppProps) {
          return <Component {...pageProps} />;
        }
        ```
      </Tab>
      <Tab value="Vite / CRA">
        ```tsx title="src/main.tsx"
        import '@anju/nuform/styles.css';
        import React from 'react';
        import ReactDOM from 'react-dom/client';
        import App from './App';

        ReactDOM.createRoot(document.getElementById('root')!).render(
          <React.StrictMode>
            <App />
          </React.StrictMode>
        );
        ```
      </Tab>
    </Tabs>

    <Callout type="warn" title="重要">
    この手順を忘れると、コンポーネントのスタイルが正しく適用されません。必ずスタイルをインポートしてください。
    </Callout>
  </Step>

  <Step>
    ### 背景色を設定

    ニューモフィズムデザインは背景色が重要です。`--nuform-bg`変数を使用して背景を設定してください。

    ```css title="globals.css"
    body {
      background: var(--nuform-bg);
      min-height: 100vh;
    }
    ```

    または、Tailwind CSSを使用している場合:

    ```tsx
    <div className="bg-(--nuform-bg) min-h-screen">
      {/* アプリケーションの内容 */}
    </div>
    ```
  </Step>

  <Step>
    ### コンポーネントを使用

    これで準備完了です！コンポーネントをインポートして使用できます。

    ```tsx title="App.tsx"
    import { Button, Card, Input, Badge, Avatar, Divider } from '@anju/nuform';

    function App() {
      return (
        <div style={{ background: 'var(--nuform-bg)', minHeight: '100vh', padding: '2rem' }}>
          <Card>
            <div style={{ display: 'flex', alignItems: 'center', gap: '1rem', marginBottom: '1rem' }}>
              <Avatar alt="User" size="lg" />
              <div>
                <h2>Welcome back!</h2>
                <Badge variant="success">Active</Badge>
              </div>
            </div>
            
            <Divider />
            
            <Input label="Email" type="email" placeholder="Enter your email" />
            
            <div style={{ marginTop: '1rem', display: 'flex', gap: '0.5rem' }}>
              <Button variant="primary">Submit</Button>
              <Button variant="ghost">Cancel</Button>
            </div>
          </Card>
        </div>
      );
    }

    export default App;
    ```
  </Step>
</Steps>

## TypeScript

<Callout type="info" title="型定義">
NuformはTypeScriptで書かれており、型定義がパッケージに含まれています。追加の`@types`パッケージは不要です。
</Callout>

コンポーネントのPropsの型をインポートできます:

```tsx
import type { ButtonProps, CardProps, InputProps, AvatarProps, BadgeProps, DividerProps } from '@anju/nuform';

// 型を使った例
const myButtonProps: ButtonProps = {
  variant: 'primary',
  size: 'lg',
  loading: false,
};
```

## CSS変数

Nuformは以下のCSS変数を提供しています。これらを上書きすることでテーマをカスタマイズできます:

<Accordions>
  <Accordion title="カラー変数">
    ```css
    :root {
      --nuform-bg: #e8eef3;           /* 背景色 */
      --nuform-text: #2d3748;         /* テキスト色 */
      --nuform-primary: #6366f1;      /* プライマリカラー */
      --nuform-shadow-light: #ffffff; /* ライトシャドウ */
      --nuform-shadow-dark: #c5ccd5;  /* ダークシャドウ */
    }
    ```
  </Accordion>
  <Accordion title="スペーシング変数">
    ```css
    :root {
      --nuform-spacing-xs: 0.25rem;
      --nuform-spacing-sm: 0.5rem;
      --nuform-spacing-md: 1rem;
      --nuform-spacing-lg: 1.5rem;
      --nuform-spacing-xl: 2rem;
    }
    ```
  </Accordion>
  <Accordion title="ボーダーラジアス変数">
    ```css
    :root {
      --nuform-radius-sm: 0.375rem;
      --nuform-radius-md: 0.5rem;
      --nuform-radius-lg: 0.75rem;
      --nuform-radius-full: 9999px;
    }
    ```
  </Accordion>
</Accordions>
