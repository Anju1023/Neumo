# Neumo UI

**Neumo UI** は、ニューモフィズム2.0デザインに基づいたモダンなReact UIコンポーネントライブラリです。

<Callout type="info" title="ニューモフィズム2.0とは？">
従来のニューモフィズムの美しさを保ちながら、**高コントラスト**と**アクセシビリティ**を重視した新しいデザインアプローチです。視認性が向上し、より多くのユーザーにとって使いやすいUIを実現します。
</Callout>

## クイックスタート

<Steps>
  <Step>
    ### パッケージをインストール

    お好みのパッケージマネージャーでインストールしてください。

    <Tabs items={['npm', 'pnpm', 'yarn', 'bun']} persist groupId="pm">
      <Tab value="npm">
        ```bash
        npm install neumo-ui
        ```
      </Tab>
      <Tab value="pnpm">
        ```bash
        pnpm add neumo-ui
        ```
      </Tab>
      <Tab value="yarn">
        ```bash
        yarn add neumo-ui
        ```
      </Tab>
      <Tab value="bun">
        ```bash
        bun add neumo-ui
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step>
    ### スタイルをインポート

    アプリケーションのエントリーポイントでスタイルをインポートします。

    ```tsx
    import 'neumo-ui/styles.css';
    ```
  </Step>

  <Step>
    ### コンポーネントを使う

    ```tsx
    import { Button, Card, Input } from 'neumo-ui';

    function App() {
      return (
        <Card>
          <h2>Welcome</h2>
          <Input placeholder="Enter your name" />
          <Button variant="primary">Submit</Button>
        </Card>
      );
    }
    ```
  </Step>
</Steps>

<Callout type="warn" title="背景色の設定を忘れずに">
ニューモフィズムデザインは背景色が重要です。`body`や親要素に`background: var(--neumo-bg)`を設定してください。
</Callout>
