# Neumo UI

**Neumo UI** は、ニューモフィズム2.0デザインに基づいたモダンなReact UIコンポーネントライブラリです。

## 特徴

- **ニューモフィズム2.0** - 高コントラストでアクセシブルなデザイン
- **React 19対応** - 最新のReact機能に対応
- **TypeScript** - 完全な型定義
- **Tree-shaking対応** - 必要なコンポーネントだけをバンドル
- **カスタマイズ可能** - CSS変数でスタイルを調整

## クイックスタート

```bash
# インストール
npm install neumo-ui

# または pnpm
pnpm add neumo-ui
```

```tsx
import { Button, Card, Input } from 'neumo-ui';
import 'neumo-ui/styles.css';

function App() {
  return (
    <Card>
      <h2>Welcome</h2>
      <Input placeholder="Enter your name" />
      <Button variant="primary">Submit</Button>
    </Card>
  );
}
```
