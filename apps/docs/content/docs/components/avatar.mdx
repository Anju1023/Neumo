# Avatar

ニューモフィズム2.0デザインのアバターコンポーネントです。ユーザーのプロフィール画像を表示し、画像がない場合はイニシャルを表示します。

## インポート

```tsx
import { Avatar } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Avatar alt="John Doe" />`}>
  <Avatar alt="John Doe" />
</ComponentPreview>

## イニシャル表示

画像がない場合、名前からイニシャルを自動生成して表示します。

<ComponentShowcase title="Avatar Initials">
  <Avatar alt="John Doe" />
  <Avatar alt="Jane" />
  <Avatar alt="Alice Bob" />
</ComponentShowcase>

| サイズ | ピクセル |
|-------|---------|
| `xs` | 24px |
| `sm` | 32px |
| `md` | 40px |
| `lg` | 56px |
| `xl` | 72px |

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `src` | `string` | - | 画像のURL |
| `alt` | `string` | **必須** | 代替テキスト（イニシャル生成にも使用） |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | アバターのサイズ |
| `variant` | `"circle" \| "rounded"` | `"circle"` | アバターの形状 |
| `fallback` | `string` | - | 画像がない場合のフォールバックテキスト |

## アクセシビリティ

- `role="img"`: 画像としての役割を明示
- `aria-label`: 代替テキストを設定
