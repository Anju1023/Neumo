# Avatar

ニューモフィズム2.0デザインのアバターコンポーネントです。ユーザーのプロフィール画像を表示し、画像がない場合はイニシャルを表示します。

<Callout type="info" title="スマートフォールバック">
画像が指定されていない場合や読み込みに失敗した場合、`alt`テキストから自動的にイニシャルを生成して表示します。
</Callout>

## インポート

```tsx
import { Avatar } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Avatar alt="John Doe" />`}>
  <Avatar alt="John Doe" />
</ComponentPreview>

## イニシャル表示

画像がない場合、名前からイニシャルを自動生成して表示します。

<ComponentShowcase title="Avatar Initials">
  <Avatar alt="John Doe" />
  <Avatar alt="Jane" />
  <Avatar alt="Alice Bob" />
</ComponentShowcase>

<Callout type="info">
- フルネーム（例: "John Doe"）→ 頭文字2文字（"JD"）
- 単一名（例: "Jane"）→ 頭文字1文字（"J"）
- 3語以上（例: "Alice Bob Carol"）→ 最初と最後の頭文字（"AC"）
</Callout>

## アクセシビリティ

<Callout type="info" title="WAI-ARIA対応">
Avatarコンポーネントは以下のアクセシビリティ機能を備えています：
</Callout>

- **`role="img"`**: 画像としての役割を明示
- **`aria-label`**: 代替テキストを設定し、スクリーンリーダーが読み上げ可能
- **高コントラスト**: イニシャル表示時も十分なコントラスト比を確保

<Callout type="warn" title="altの重要性">
`alt`プロパティは必須です。これはイニシャル生成とアクセシビリティの両方に使用されます。ユーザーを識別できる意味のあるテキストを設定してください。
</Callout>
