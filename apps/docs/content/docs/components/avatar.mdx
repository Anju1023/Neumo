# Avatar

ニューモフィズム2.0デザインのアバターコンポーネントです。ユーザーのプロフィール画像を表示し、画像がない場合はイニシャルを表示します。

## インポート

```tsx
import { Avatar } from 'neumo-ui';
```

## 基本的な使い方

```tsx
<Avatar src="/avatar.jpg" alt="John Doe" />
```

## イニシャル表示

画像がない場合や読み込みに失敗した場合、名前からイニシャルを自動生成して表示します。

```tsx
<Avatar alt="John Doe" />           {/* JD */}
<Avatar alt="Jane" />               {/* J */}
<Avatar alt="John Michael Doe" />   {/* JD */}
```

## カスタムフォールバック

```tsx
<Avatar alt="User" fallback="?" />
```

## サイズ

5種類のサイズがあります。

```tsx
<Avatar alt="User" size="xs" />  {/* 24px */}
<Avatar alt="User" size="sm" />  {/* 32px */}
<Avatar alt="User" size="md" />  {/* 40px */}
<Avatar alt="User" size="lg" />  {/* 56px */}
<Avatar alt="User" size="xl" />  {/* 72px */}
```

## バリアント（形状）

```tsx
<Avatar alt="User" variant="circle" />   {/* 円形（デフォルト） */}
<Avatar alt="User" variant="rounded" />  {/* 角丸四角形 */}
```

## 画像付き

```tsx
<Avatar 
  src="https://example.com/avatar.jpg" 
  alt="John Doe" 
  size="lg"
  variant="circle"
/>
```

## アバターグループ

複数のアバターを重ねて表示する例です。

```tsx
<div style={{ display: 'flex' }}>
  <Avatar alt="User 1" style={{ marginRight: '-8px', zIndex: 3 }} />
  <Avatar alt="User 2" style={{ marginRight: '-8px', zIndex: 2 }} />
  <Avatar alt="User 3" style={{ zIndex: 1 }} />
</div>
```

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `src` | `string` | - | 画像のURL |
| `alt` | `string` | **必須** | 代替テキスト（イニシャル生成にも使用） |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | アバターのサイズ |
| `variant` | `"circle" \| "rounded"` | `"circle"` | アバターの形状 |
| `fallback` | `string` | - | 画像がない場合のフォールバックテキスト |

## アクセシビリティ

- `role="img"`: 画像としての役割を明示
- `aria-label`: 代替テキストを設定
