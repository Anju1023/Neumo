# Input

ニューモフィズム2.0デザインの入力フィールドコンポーネントです。

<Callout type="info" title="デザインコンセプト">
凹んだ（inset）デザインで、フォーカス時にさらに深くなるエフェクトがあります。これにより、ユーザーは入力フィールドを直感的に認識できます。
</Callout>

## インポート

```tsx
import { Input } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Input placeholder="Enter text..." />`}>
  <NeumoInput placeholder="Enter text..." />
</ComponentPreview>

## ラベル付き

`label`プロパティでラベルを表示できます。ラベルは自動的に入力フィールドと関連付けられます。

<ComponentPreview code={`<Input label="Email" placeholder="Enter your email" />`}>
  <NeumoInput label="Email" placeholder="Enter your email" />
</ComponentPreview>

## ヘルパーテキスト

`helperText`プロパティで補足説明を表示できます。

<ComponentPreview code={`<Input 
  label="Email" 
  placeholder="example@email.com"
  helperText="We'll never share your email"
/>`}>
  <NeumoInput 
    label="Email" 
    placeholder="example@email.com"
    helperText="We'll never share your email"
  />
</ComponentPreview>

## エラー状態

バリデーションエラーを表示できます。

<ComponentPreview code={`<Input 
  label="Email" 
  error 
  errorMessage="Invalid email address"
/>`}>
  <NeumoInput 
    label="Email" 
    error 
    errorMessage="Invalid email address"
  />
</ComponentPreview>

<Callout type="warn" title="エラーメッセージ">
`error`プロパティを`true`にすると、入力フィールドが赤くハイライトされます。`errorMessage`を指定すると、エラーの詳細が表示されます。
</Callout>

## アクセシビリティ

<Callout type="info" title="WAI-ARIA対応">
Inputコンポーネントは以下のアクセシビリティ機能を備えています：
</Callout>

- **`aria-invalid`**: エラー時に`true`に設定
- **`aria-describedby`**: エラーメッセージまたはヘルパーテキストを関連付け
- **ラベル関連付け**: `label`要素と`input`要素が`htmlFor`/`id`で正しく関連付け
- **フォーカスリング**: キーボードナビゲーション時に視認性の高いフォーカスリングを表示
- **適切なコントラスト**: プレースホルダーテキストも読みやすい色で設計

<Callout type="warn" title="ラベルの重要性">
アクセシビリティの観点から、入力フィールドには常に`label`を設定することを推奨します。視覚的に非表示にしたい場合でも、スクリーンリーダー用に設定してください。
</Callout>
