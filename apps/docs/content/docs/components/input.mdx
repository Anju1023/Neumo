# Input

ニューモフィズム2.0デザインの入力フィールドコンポーネントです。凹んだ（inset）デザインで、フォーカス時に深くなるエフェクトがあります。

## インポート

```tsx
import { Input } from 'neumo-ui';
```

## 基本的な使い方

```tsx
<Input placeholder="Enter text..." />
```

## ラベル付き

```tsx
<Input label="Email" placeholder="Enter your email" />
```

## タイプ

5種類の入力タイプに対応しています。

```tsx
<Input type="text" label="Text" />
<Input type="email" label="Email" />
<Input type="password" label="Password" />
<Input type="number" label="Number" />
<Input type="search" label="Search" />
```

## サイズ

3種類のサイズがあります。

```tsx
<Input size="sm" placeholder="Small" />
<Input size="md" placeholder="Medium" />
<Input size="lg" placeholder="Large" />
```

## ヘルパーテキスト

```tsx
<Input 
  label="Email" 
  placeholder="example@email.com"
  helperText="We'll never share your email"
/>
```

## エラー状態

```tsx
<Input 
  label="Email" 
  error 
  errorMessage="Invalid email address"
/>
```

## アイコン付き

左右にアイコンを配置できます。

```tsx
import { Mail, Search, Eye } from 'lucide-react';

<Input leftIcon={<Mail size={16} />} placeholder="Email" />
<Input leftIcon={<Search size={16} />} placeholder="Search..." />
<Input type="password" rightIcon={<Eye size={16} />} placeholder="Password" />
```

## 無効状態

```tsx
<Input disabled placeholder="Disabled input" />
```

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `type` | `"text" \| "password" \| "email" \| "number" \| "search"` | `"text"` | 入力タイプ |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 入力フィールドのサイズ |
| `error` | `boolean` | `false` | エラー状態 |
| `errorMessage` | `string` | - | エラーメッセージ |
| `leftIcon` | `ReactNode` | - | 左側に表示するアイコン |
| `rightIcon` | `ReactNode` | - | 右側に表示するアイコン |
| `label` | `string` | - | ラベルテキスト |
| `helperText` | `string` | - | ヘルパーテキスト |
| `disabled` | `boolean` | `false` | 無効状態 |

## アクセシビリティ

- `aria-invalid`: エラー時に`true`に設定
- `aria-describedby`: エラーメッセージまたはヘルパーテキストを関連付け
- ラベルとinputの関連付け（`htmlFor`/`id`）
- フォーカス時にフォーカスリングを表示
