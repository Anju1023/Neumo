# Input

ニューモフィズム2.0デザインの入力フィールドコンポーネントです。凹んだ（inset）デザインで、フォーカス時に深くなるエフェクトがあります。

## インポート

```tsx
import { Input } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Input placeholder="Enter text..." />`}>
  <Input placeholder="Enter text..." />
</ComponentPreview>

<ComponentPreview code={`<Input label="Email" placeholder="Enter your email" />`}>
  <Input label="Email" placeholder="Enter your email" />
</ComponentPreview>

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `type` | `"text" \| "password" \| "email" \| "number" \| "search"` | `"text"` | 入力タイプ |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 入力フィールドのサイズ |
| `error` | `boolean` | `false` | エラー状態 |
| `errorMessage` | `string` | - | エラーメッセージ |
| `leftIcon` | `ReactNode` | - | 左側に表示するアイコン |
| `rightIcon` | `ReactNode` | - | 右側に表示するアイコン |
| `label` | `string` | - | ラベルテキスト |
| `helperText` | `string` | - | ヘルパーテキスト |
| `disabled` | `boolean` | `false` | 無効状態 |

## アクセシビリティ

- `aria-invalid`: エラー時に`true`に設定
- `aria-describedby`: エラーメッセージまたはヘルパーテキストを関連付け
- ラベルとinputの関連付け（`htmlFor`/`id`）
- フォーカス時にフォーカスリングを表示
