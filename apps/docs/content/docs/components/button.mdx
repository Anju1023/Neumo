# Button

ニューモフィズム2.0デザインのボタンコンポーネントです。

<Callout type="info" title="インタラクションエフェクト">
ホバーで**浮き上がり**、クリックで**押し込まれる**エフェクトがあります。この立体的なフィードバックにより、ユーザーは操作を直感的に理解できます。
</Callout>

## インポート

```tsx
import { Button } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Button>Default Button</Button>`}>
  <NeumoButton>Default Button</NeumoButton>
</ComponentPreview>

## ローディング状態

非同期処理中などにローディング状態を表示できます。

<ComponentPreview code={`<Button loading>Loading...</Button>`}>
  <NeumoButton loading>Loading...</NeumoButton>
</ComponentPreview>

<Callout type="info">
ローディング中は自動的に`aria-busy="true"`が設定され、クリックできなくなります。
</Callout>

## アクセシビリティ

<Callout type="info" title="WAI-ARIA対応">
Buttonコンポーネントは以下のアクセシビリティ機能を備えています：
</Callout>

- **`aria-busy`**: ローディング時に`true`に設定され、スクリーンリーダーに処理中であることを伝えます
- **`aria-disabled`**: 無効時に`true`に設定されます
- **フォーカスリング**: キーボードフォーカス時に視認性の高いフォーカスリングを表示
- **適切なコントラスト**: WCAG 2.1 AAレベルのコントラスト比を満たすカラー設計

<Callout type="warn" title="注意">
`ghost`バリアントを使用する場合は、背景色とのコントラストが十分であることを確認してください。
</Callout>
