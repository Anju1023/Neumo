# Button

ニューモフィズム2.0デザインのボタンコンポーネントです。ホバーで浮き上がり、クリックで押し込まれるエフェクトがあります。

## インポート

```tsx
import { Button } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Button>Default Button</Button>`}>
  <Button>Default Button</Button>
</ComponentPreview>

## バリアント

3種類のバリアントがあります。

<ComponentShowcase title="Button Variants">
  <Button variant="default">Default</Button>
  <Button variant="primary">Primary</Button>
  <Button variant="ghost">Ghost</Button>
</ComponentShowcase>

| バリアント | 説明 |
|-----------|------|
| `default` | 標準のニューモフィズムボタン |
| `primary` | プライマリカラーのボタン |
| `ghost` | 背景なしの透明ボタン |

## サイズ

3種類のサイズがあります。

<ComponentShowcase title="Button Sizes">
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</ComponentShowcase>

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `variant` | `"default" \| "primary" \| "ghost"` | `"default"` | ボタンのスタイルバリアント |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | ボタンのサイズ |
| `loading` | `boolean` | `false` | ローディング状態 |
| `disabled` | `boolean` | `false` | 無効状態 |
| `leftIcon` | `ReactNode` | - | 左側に表示するアイコン |
| `rightIcon` | `ReactNode` | - | 右側に表示するアイコン |
| `children` | `ReactNode` | - | ボタンの中身 |

## アクセシビリティ

- `aria-busy`: ローディング時に`true`に設定
- `aria-disabled`: 無効時に`true`に設定
- キーボードフォーカス時にフォーカスリングを表示
