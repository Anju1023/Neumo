# Badge

ニューモフィズム2.0デザインのバッジコンポーネントです。ステータスやカテゴリを表示するための小さなラベルとして使用します。

## インポート

```tsx
import { Badge } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Badge>Default</Badge>`}>
  <Badge>Default</Badge>
</ComponentPreview>

## バリアント

5種類のバリアントがあります。

<ComponentShowcase title="Badge Variants">
  <Badge variant="default">Default</Badge>
  <Badge variant="success">Success</Badge>
  <Badge variant="warning">Warning</Badge>
  <Badge variant="error">Error</Badge>
  <Badge variant="info">Info</Badge>
</ComponentShowcase>

| バリアント | 説明 | 使用例 |
|-----------|------|--------|
| `default` | 標準バッジ | 一般的なラベル |
| `success` | 成功・完了状態 | 「完了」「アクティブ」 |
| `warning` | 警告状態 | 「保留中」「注意」 |
| `error` | エラー状態 | 「エラー」「失敗」 |
| `info` | 情報表示 | 「新着」「ベータ」 |

## サイズ

2種類のサイズがあります。

<ComponentShowcase title="Badge Sizes">
  <Badge size="sm">Small</Badge>
  <Badge size="md">Medium</Badge>
</ComponentShowcase>

## 使用例

</div>`}>

    <Badge variant="success">Online</Badge>
    <Badge variant="warning">Away</Badge>
    <Badge variant="default">Offline</Badge>

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `variant` | `"default" \| "success" \| "warning" \| "error" \| "info"` | `"default"` | バッジのスタイルバリアント |
| `size` | `"sm" \| "md"` | `"md"` | バッジのサイズ |
| `children` | `ReactNode` | - | バッジの中身 |
