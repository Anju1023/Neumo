# Card

ニューモフィズム2.0デザインのカードコンポーネントです。コンテンツを囲むためのコンテナとして使用します。

## インポート

```tsx
import { Card } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Card>
  <h2>Card Title</h2>
  <p>Card content goes here</p>
</Card>`}>
  <Card>
    <h2 style={{ margin: 0, marginBottom: '0.5rem' }}>Card Title</h2>
    <p style={{ margin: 0, color: '#718096' }}>Card content goes here</p>
  </Card>
</ComponentPreview>

## バリアント

3種類のバリアントがあります。

<ComponentShowcase title="Card Variants">
  <Card variant="elevated" style={{ width: '200px' }}>
    <h4 style={{ margin: 0, marginBottom: '0.5rem' }}>Elevated</h4>
    <p style={{ margin: 0, fontSize: '0.875rem', color: '#718096' }}>浮き上がったカード</p>
  </Card>
  <Card variant="flat" style={{ width: '200px' }}>
    <h4 style={{ margin: 0, marginBottom: '0.5rem' }}>Flat</h4>
    <p style={{ margin: 0, fontSize: '0.875rem', color: '#718096' }}>フラットなカード</p>
  </Card>
  <Card variant="inset" style={{ width: '200px' }}>
    <h4 style={{ margin: 0, marginBottom: '0.5rem' }}>Inset</h4>
    <p style={{ margin: 0, fontSize: '0.875rem', color: '#718096' }}>凹んだカード</p>
  </Card>
</ComponentShowcase>

| バリアント | 説明 |
|-----------|------|
| `elevated` | 浮き上がったカード（デフォルト） |
| `flat` | フラットなカード（シャドウなし） |
| `inset` | 凹んだカード |

## パディング

4種類のパディングサイズがあります。

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `variant` | `"elevated" \| "flat" \| "inset"` | `"elevated"` | カードのスタイルバリアント |
| `padding` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | カードのパディング |
| `children` | `ReactNode` | - | カードの中身 |
