# Card

ニューモフィズム2.0デザインのカードコンポーネントです。コンテンツを視覚的にグループ化するためのコンテナとして使用します。

<Callout type="info" title="デザインコンセプト">
Cardは背景から**浮き上がった**または**凹んだ**外観を持ち、コンテンツの階層構造を明確にします。3種類のバリアントでさまざまな表現が可能です。
</Callout>

## インポート

```tsx
import { Card } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<Card>
  <h2>Card Title</h2>
  <p>Card content goes here</p>
</Card>`}>
  <Card>
    <h2 style={{ margin: 0, marginBottom: '0.5rem' }}>Card Title</h2>
    <p style={{ margin: 0, color: '#718096' }}>Card content goes here</p>
  </Card>
</ComponentPreview>

## バリアント

3種類のバリアントがあります。コンテンツの重要度や用途に応じて使い分けてください。

    <p style={{ margin: 0, fontSize: '0.875rem', color: '#718096' }}>浮き上がったカード</p>
  </Card>
  <Card variant="flat" style={{ width: '200px' }}>

    <h4 style={{ margin: 0, marginBottom: '0.5rem' }}>Inset</h4>

  </Card>
</ComponentShowcase>

### ネストされたカード

`inset`バリアントを使って、カード内にネストした領域を作成できます。

<ComponentPreview code={`<Card variant="elevated" padding="lg">
  <h3>Settings</h3>
  <Card variant="inset" padding="md">
    <p>Nested inset card for secondary content</p>
  </Card>
</Card>`}>
  <Card variant="elevated" padding="lg" style={{ width: '320px' }}>
    <h3 style={{ margin: '0 0 1rem' }}>Settings</h3>
    <Card variant="inset" padding="md">
      <p style={{ margin: 0, color: '#718096', fontSize: '0.875rem' }}>Nested inset card for secondary content</p>
    </Card>
  </Card>
</ComponentPreview>

## アクセシビリティ

<Callout type="info" title="セマンティックHTML">
Cardコンポーネントは`<div>`要素でレンダリングされます。コンテンツの構造に応じて、適切な見出しレベル（h1〜h6）を使用してください。
</Callout>

<Callout type="warn" title="インタラクティブなカード">
カード全体をクリック可能にする場合は、内部にボタンやリンクを配置するか、カード自体を`<a>`や`<button>`でラップしてください。その際、適切な`role`と`tabIndex`を設定してください。
</Callout>
