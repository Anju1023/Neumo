# Card

ニューモフィズム2.0デザインのカードコンポーネントです。コンテンツを囲むためのコンテナとして使用します。

## インポート

```tsx
import { Card } from 'neumo-ui';
```

## 基本的な使い方

```tsx
<Card>
  <h2>Card Title</h2>
  <p>Card content goes here</p>
</Card>
```

## バリアント

3種類のバリアントがあります。

```tsx
<Card variant="elevated">
  <h3>Elevated Card</h3>
  <p>浮き上がったカード（デフォルト）</p>
</Card>

<Card variant="flat">
  <h3>Flat Card</h3>
  <p>フラットなカード</p>
</Card>

<Card variant="inset">
  <h3>Inset Card</h3>
  <p>凹んだカード</p>
</Card>
```

| バリアント | 説明 |
|-----------|------|
| `elevated` | 浮き上がったカード（デフォルト） |
| `flat` | フラットなカード（シャドウなし） |
| `inset` | 凹んだカード |

## パディング

4種類のパディングサイズがあります。

```tsx
<Card padding="none">No padding</Card>
<Card padding="sm">Small padding</Card>
<Card padding="md">Medium padding</Card>
<Card padding="lg">Large padding</Card>
```

## 組み合わせ例

```tsx
<Card variant="elevated" padding="lg">
  <div style={{ display: 'flex', alignItems: 'center', gap: '1rem' }}>
    <Avatar alt="User" size="lg" />
    <div>
      <h3>John Doe</h3>
      <Badge variant="success">Online</Badge>
    </div>
  </div>
  <Divider />
  <p>This is a profile card example combining multiple components.</p>
  <Button variant="primary">View Profile</Button>
</Card>
```

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `variant` | `"elevated" \| "flat" \| "inset"` | `"elevated"` | カードのスタイルバリアント |
| `padding` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | カードのパディング |
| `children` | `ReactNode` | - | カードの中身 |
