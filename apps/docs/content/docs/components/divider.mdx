# Divider

ニューモフィズム2.0デザインの区切り線コンポーネントです。溝（groove）のような凹んだデザインで、コンテンツを視覚的に分割します。

## インポート

```tsx
import { Divider } from 'neumo-ui';
```

## 基本的な使い方

```tsx
<Divider />
```

## 方向

水平と垂直の2つの方向があります。

### 水平方向（デフォルト）

```tsx
<div>
  <p>Content above</p>
  <Divider orientation="horizontal" />
  <p>Content below</p>
</div>
```

### 垂直方向

```tsx
<div style={{ display: 'flex', alignItems: 'center', height: '40px' }}>
  <span>Left</span>
  <Divider orientation="vertical" />
  <span>Right</span>
</div>
```

## 使用例

### カード内での使用

```tsx
<Card>
  <h3>Section 1</h3>
  <p>First section content</p>
  <Divider />
  <h3>Section 2</h3>
  <p>Second section content</p>
</Card>
```

### リストの区切り

```tsx
<div>
  <div style={{ padding: '1rem' }}>
    <h4>Item 1</h4>
    <p>Description</p>
  </div>
  <Divider />
  <div style={{ padding: '1rem' }}>
    <h4>Item 2</h4>
    <p>Description</p>
  </div>
  <Divider />
  <div style={{ padding: '1rem' }}>
    <h4>Item 3</h4>
    <p>Description</p>
  </div>
</div>
```

### ナビゲーション区切り

```tsx
<nav style={{ display: 'flex', alignItems: 'center', gap: '1rem' }}>
  <a href="/">Home</a>
  <Divider orientation="vertical" />
  <a href="/about">About</a>
  <Divider orientation="vertical" />
  <a href="/contact">Contact</a>
</nav>
```

## スタイルのカスタマイズ

```tsx
<Divider 
  style={{ 
    margin: '2rem 0',
    width: '50%' 
  }} 
/>
```

## Props

| Prop | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | 区切り線の方向 |

## アクセシビリティ

- `role="separator"`: セパレーターとしての役割を明示
- `aria-orientation`: 方向を明示
