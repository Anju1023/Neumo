# Divider

ニューモフィズム2.0デザインの区切り線コンポーネントです。コンテンツを視覚的に分割するために使用します。

<Callout type="info" title="デザインコンセプト">
溝（groove）のような凹んだデザインで、ニューモフィズムの背景に自然に溶け込みながら、コンテンツの区切りを明確にします。
</Callout>

## インポート

```tsx
import { Divider } from 'neumo-ui';
```

## 基本的な使い方

<ComponentPreview code={`<div style={{ width: '100%' }}>
  <p>Content above</p>
  <Divider />
  <p>Content below</p>
</div>`}>
  <div style={{ width: '300px' }}>
    <p style={{ margin: '0 0 1rem' }}>Content above</p>
    <Divider />
    <p style={{ margin: '1rem 0 0' }}>Content below</p>
  </div>
</ComponentPreview>

## 方向

水平と垂直の2つの方向があります。

<Tabs items={['水平方向', '垂直方向']}>
  <Tab value="水平方向">
    水平方向（horizontal）はデフォルトです。上下のコンテンツを分離します。

    <ComponentPreview code={`<Divider orientation="horizontal" />`}>
      <div style={{ width: '200px' }}>
        <Divider orientation="horizontal" />
      </div>
    </ComponentPreview>
  </Tab>
  <Tab value="垂直方向">
    垂直方向（vertical）は左右のコンテンツを分離します。Flexboxコンテナ内で使用してください。

    <ComponentPreview code={`<div style={{ display: 'flex', alignItems: 'center', height: '40px', gap: '1rem' }}>
  <span>Left</span>
  <Divider orientation="vertical" />
  <span>Right</span>
</div>`}>
      <div style={{ display: 'flex', alignItems: 'center', height: '40px', gap: '1rem' }}>
        <span>Left</span>
        <Divider orientation="vertical" />
        <span>Right</span>
      </div>
    </ComponentPreview>
  </Tab>
</Tabs>

<Callout type="warn" title="垂直方向の注意点">
`orientation="vertical"`を使用する場合、親要素に`height`が設定されている必要があります。また、親要素は`display: flex`と`align-items: center`または`stretch`を設定してください。
</Callout>

## アクセシビリティ

<Callout type="info" title="セマンティックHTML">
Dividerコンポーネントは以下のアクセシビリティ機能を備えています：
</Callout>

- **`role="separator"`**: セパレーターとしての役割を明示し、スクリーンリーダーがコンテンツの区切りを認識
- **`aria-orientation`**: 方向（horizontal/vertical）を明示

<Callout type="info" title="装飾的な区切り線">
純粋に装飾目的で使用する場合は、`aria-hidden="true"`を追加することを検討してください。ただし、コンテンツの論理的な区切りを示す場合は、そのままにしてください。
</Callout>

    - マージンやパディングの代わりに使用

    - 垂直方向で使用する場合、親要素のFlexbox設定を確認
    - 余白は親要素側で調整（Divider自体にマージンを追加しない）
    - 色やサイズの変更はCSS変数で行う
  </Accordion>
</Accordions>
